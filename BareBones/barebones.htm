Bare Bones Simulator

A very simple java simulator for the Bare Bones programming language. Needs Java 6 to run.

Commands

  • clear x; - clears a variable called x to 0
  • incr x; - adds 1 to a variable called x
  • decr x; - subtracts 1 from a variable called x
  • while x; - creates a loop that runs while x > 0
  • end; - marks the end of a while loop
  • copy x to y; - copies the value of variable x to variable y
  • set x to 5; - sets a variable called x to the value 5.
